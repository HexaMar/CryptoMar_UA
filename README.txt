## Заборонені Дії

Користувач програми CryptoMar зобов'язується не використовувати програмне забезпечення для:

* Зняття коштів із згенерованих Bitcoin-гаманців без правомірного доступу.
* Переведення коштів на або з згенерованих гаманців без згоди власника коштів.
* Вчинення будь-яких операцій з Bitcoin-адресами, створеними програмою, якщо це порушує законодавство країни користувача.

Програма CryptoMar використовує алгоритми випадкової генерації для створення Bitcoin-адрес та приватних ключів. Всі адреси для комісій та інші внутрішні дані генеруються випадковим чином і не зберігаються.

Користувач усвідомлює, що будь-які дії, вчинені з використанням програми CryptoMar, можуть мати юридичні наслідки. Використання програми для доступу до коштів, що належать третім особам, без їх згоди є незаконним і може призвести до кримінальної або адміністративної відповідальності.

Програма CryptoMar створена виключно для освітніх цілей. Вивчивши принципи роботи програми, користувач може дізнатися основи генерації Bitcoin-адрес та принцип роботи криптографічних алгоритмів. Подібним чином може виглядати реальна програма для створення та управління криптовалютними гаманцями.
